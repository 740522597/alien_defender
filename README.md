# Alien Defender

浏览器摄像头体感射击原型。玩家举手后，炮塔会朝手部光标连续射击，拦截外星舰队、敌弹、武器包和医疗舱。

## 运行

```bash
python3 -m http.server 8899
```

打开：

```text
http://127.0.0.1:8899/
```

摄像头需要通过 `localhost`、`127.0.0.1` 或 `https` 打开页面，直接双击 `index.html` 通常无法访问摄像头。

## 当前内容

- MediaPipe Hands 识别手部位置。
- 单手控制双炮塔，双手时左右炮塔分别跟随对应手。
- 10 个关卡，支持简单、中等、困难三种难度。
- 外星舰队包含侦察机、截击机、突击艇、重甲舰、飞碟和母舰。
- 敌舰会预警后向底部防线开火。
- 玩家子弹可以击毁敌舰，也可以拦截敌弹。
- 护盾先承受伤害，短时间未受伤后自动恢复。
- 武器包可临时切换机枪、散弹、激光、火焰、双发等武器。
- 医疗舱可恢复生命值。
- 每关结束必须选择一个升级后才进入下一关。

## 文件

```text
index.html
style.css
main.js
assets/alien_sprites.png
README.md
```

`assets/alien_sprites.png` 是 3x3 排版的敌舰精灵图，当前代码按以下位置裁切：侦察机、截击机、突击艇、重甲舰、飞碟、母舰、武器补给舱、医疗舱。缺少该文件时，游戏会使用 Canvas 绘制的旧外观。

## 常见问题

- 摄像头失败：确认浏览器摄像头权限已允许，并关闭其他占用摄像头的软件。
- 无法识别手：手掌尽量完整出现在画面中，并保持光线充足。
- 页面没有更新：浏览器强制刷新，或确认页面已加载最新 `main.js` 版本参数。
